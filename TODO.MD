# TeacherStory — План разработки

## Идея

Фото-портфолио для учителя с 30+ летним стажем. Хронологическая история карьеры, школьных событий, выпусков. Часть альбомов публичная, часть — платная (card-to-card).

## Доноры

- **VDole** (`../VDole/`) — auth (VK, Telegram, MAX), CAS storage, DB patterns, frontend scaffold
- **iamrich** (`../../iamrich/`) — платежи (card-to-card, скриншоты, AI-парсинг)
